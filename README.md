**Création d'un site de tatouage**
  ==============================

Le projet de la semaine06 consisté à créer un site de tatouage contenant quatre pages:
	Une page d'accueil avec un background sans scroll.
	Une page de contact avec un formulaire de contact et une map.
	Une page avec une galerie d'image.
	Une page blog avec des articles concernant le monde du tatouage.

Avec ceci on devait créer un menu qui devait rester fixe c'est à dire que même si la page descendait le menu restait toujours à la même place sur l'écran.
